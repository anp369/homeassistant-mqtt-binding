homeassistant-mqtt-binding documentation!
======================================================

**What is this all about?**

This library acts as an bridge between python and homeassistant.
It provides python interfaces for some useful devices such as switches or
sensors and uses mqtt and the homeassistant auto discovery mechanism
to inject those devices into your homeassistant installation.
